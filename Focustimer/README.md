# Focus Timer (Pomodoro Timer)

A simple productivity timer based on the Pomodoro technique that helps users stay focused while studying or working.

## Features

* 25 minute focus session
* 5 minute break timer
* Start / Pause / Reset controls
* Clean dark theme UI

## Technologies Used

* HTML
* CSS
* JavaScript

## How It Works

The timer uses JavaScript `setInterval()` to track time and automatically switches between focus and break sessions.
