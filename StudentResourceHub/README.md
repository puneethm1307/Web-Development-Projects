# Student Resource Hub

A multi-page website designed to provide useful learning resources and tools for students.

## Pages Included

* Home
* Resources
* Tools
* Contact

## Features

* Navigation bar for easy browsing
* Study resource cards
* Useful developer tools
* Contact form UI

## Technologies Used

* HTML
* CSS
* JavaScript

## Purpose

The goal of this project is to create a central place where students can access educational resources and development tools.

## Future Improvements

* Add downloadable study materials
* Add search functionality
* Add authentication system
