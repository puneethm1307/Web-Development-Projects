# GitHub Profile Analyzer

A web application that analyzes and displays GitHub user profile information using the GitHub API.

## Features

* Search GitHub users by username
* Display profile picture
* Show followers and following count
* Show number of public repositories
* Direct link to GitHub profile
* Modern gradient UI

## Technologies Used

* HTML
* CSS
* JavaScript
* GitHub REST API
* Fetch API

## How It Works

The application sends a request to the GitHub API and dynamically displays the returned user profile data.
