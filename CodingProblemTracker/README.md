# Coding Problem Tracker

A web application that helps developers track solved coding problems and their difficulty level.

## Features

* Add coding problems
* Select difficulty (Easy / Medium / Hard)
* Delete solved problems
* Data stored using localStorage

## Technologies Used

* HTML
* CSS
* JavaScript
* Local Storage API

## Use Case

Useful for tracking progress while practicing problems on platforms like LeetCode, HackerRank, or CodeChef.

## Future Improvements

* Add problem links
* Add solved date
* Add difficulty statistics
