# Habit Tracker

A productivity dashboard that helps users track daily habits and maintain consistency.

## Features

* Add new habits
* Mark habits as completed
* Habit streak counter
* Progress indicator
* Modern dashboard UI

## Technologies Used

* HTML
* CSS
* JavaScript
* Local Storage API

## Purpose

Helps users build good habits like coding, exercising, or reading.

## Future Improvements

* Weekly progress charts
* Habit completion calendar
* Reminder notifications
